Session Alive Installed

Quick Start

  1. Sign-in to your website to navigate to the logged in page
  2. Click the Session Alive extension icon in browser toolbar or extension menu or overflow menu.
  3. Click the button.
  4. Page is reloaded and Your session is now alive!

How to use

  1. Sign-in to your website and Copy the URL
  2. You can click the button to quickly create a rule.
  3. Click the to manage the rules.
  4. Click the button to add a New Rule and paste the copied URL to Background Request Options → Trigger URL
  5. Set the Request Interval as per your needs and click the button to save.
  6. Click the Trigger URL link in the Save Success message. Navigate to Trigger URL to start the Rule.
  7. You can also Refresh the Dashboard/Welcome URL or Sign-in again
  8. Your session is now alive!

Session Alive does not collect any user information or data.
